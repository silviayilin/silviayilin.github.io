Lazy loading 延遲載入

發表於 2021-12-14 更新於 2021-12-16 分類於 Angular
前言

目前工作是在銀行專案，初期尚未導入延遲載入，
隨著開發的模組(NgModule)達到千單位，初始載入時間至少超過一分鐘，
團隊透過延遲載入结合路由配置，在需要時才動態加載相應模組以解決網站開啟太慢的問題。

實作

建立專案

建立branch延遲載入路由模組

宣告的路由 branch 新增到以 --module 選項指定的模組中宣告的 routes 陣列中。

使用 Chrome 開發者工具，將初始載入的資訊先清除

測試點擊 Branch 來確認一下branch module真的是延遲載入的。
